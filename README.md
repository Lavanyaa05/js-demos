# JavaScript Fundamentals Demos

I made three small projects to practise JavaScript fundamentals.

## Projects

### Score Keeper

A score counter that can increase, decrease and reset.

Concepts used:

- scope
- closures
- hoisting
- DOM manipulation
- click events

### To-Do List

A task list where tasks can be added, completed and deleted.

Concepts used:

- arrays
- objects
- functions
- DOM manipulation
- form and click events

### GitHub Profile Finder

Searches for a GitHub user and displays their public profile information.

Concepts used:

- Fetch API
- Promises
- `.then()`
- `.catch()`
- JSON
- error handling
